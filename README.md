Technologies I used -

- [Next.js]

- [VS Code] with [ESLint], [EditorConfig] and [Prettier RC]

## Getting started

### Requirements

- [Node.js]
- [Yarn] or [npm]

**Access the folder**

```bash
$ cd login-nextjs
```

**Follow the steps below**

**Running application**

```bash
# Install the dependencies
$ yarn

# Run the web app
$ yarn dev
```
